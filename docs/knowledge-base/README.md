# Knowledge Base

This section is the operator-facing knowledge base for Spiderweb.

It is meant to answer practical questions such as:
- how to install Spiderweb
- how to start it and inspect its state
- where runtime and self-care files live
- how to troubleshoot common failures
- how observer, self-care, and dashboard access are expected to work
